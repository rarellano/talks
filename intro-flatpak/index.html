¿Que es Flatpak?

  • Apps empaquetadas independientes de la distribución
  • Entornos consistentes
  • Apps en sandboxes
  • Pensado para el escritorio

¿Como funciona?

Añadir repo:

            
            
$ flatpak remote-add flathub https://dl.flathub.org/repo/flathub.flatpakrepo

Busca!

            
            
$ flatpak search gimp
Application ID Version Branch Remotes Description                        
org.gimp.GIMP          stable flathub Cree imágenes y edite fotografías

Instalar y desinstalar:

            
            
$ flatpak install flathub gimp
$ flatpak uninstall gimp
